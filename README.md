# 💪 운동 부위 가이드

[`https://naryeo628.github.io/workout/`](https://naryeo628.github.io/workout/)

인체 그림에서 부위를 누르면 그 부위 운동·기구·방법을 보여주는 운동 레퍼런스. 빌드 없음(정적 HTML 1파일).

## 기능
- **인체도 앞·뒤 동시 표시** (12부위: 가슴·등·어깨·이두·삼두·전완·복근·승모·대퇴사두·햄스트링·둔근·종아리)
  - 부위 탭 → 그 부위 쓰는 운동 필터
  - 운동 카드 **hover**(데스크톱) 또는 **클릭 선택**(모바일) → 주동근(빨강)·협응근(주황) 인체도에 표시
  - 인체도 **sticky** — 카드 스크롤해도 따라옴
- **기구 필터** — 바벨·덤벨·머신·케이블·맨몸·케틀벨·밴드·스미스
- **검색**, 카드의 **방법 보기 버튼** → 단계별 방법·팁 모달
- 반응형 (모바일/데스크톱)

운동 52종 수록. `EX` 배열에 `{ n, p(주부위), s[보조], e[기구], lv, steps[], tip }` 추가하면 확장.
