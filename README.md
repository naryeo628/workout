# 💪 운동 부위 가이드

[`https://naryeo628.github.io/workout/`](https://naryeo628.github.io/workout/)

인체 그림에서 부위를 누르면 그 부위 운동·기구·방법을 보여주는 운동 레퍼런스. 빌드 없음(정적 HTML 1파일).

## 기능
- **클릭 인체도** (앞/뒤) — 가슴·등·어깨·이두·삼두·전완·복근·승모·대퇴사두·햄스트링·둔근·종아리 12부위
  - 부위 탭 → 주동근/협응근으로 그 부위를 쓰는 운동 자동 필터 + 하이라이트
- **기구 필터** — 바벨·덤벨·머신·케이블·맨몸·케틀벨·밴드·스미스
- **검색**, 운동 카드 (부위·기구·난이도 태그)
- **상세 모달** — 타겟 부위(주동/협응), 기구, 단계별 방법, 팁

운동 42종 수록. `EX` 배열에 `{ n, p(주부위), s[보조], e[기구], lv, steps[], tip }` 추가하면 확장.
